# 个人博客管理系统	

### 1.技术使用

```
前端: vue vue-cli element-ui
后端:ssm
```

### 2.前端页面: 

```
分页查看所有的博客
快速查看博客数最多的6个分类
查看所有的分类
查看某个分类下的博客列表
快速查看标记博客最多的10个标签
查看所有的标签
查看某个标签下的博客列表
根据年度时间线查看博客列表
快速查看最新的推荐博客
用关键字全局搜索博客
查看单个博客内容
对博客内容进行评论
赞赏博客内容
```

### 3.后台管理:

```
管理员 :头像 用户名 密码 生日 年龄 性别 qq github
	登录
	修改个人信息
	
管理博客 :博文ID、发布日期、发表用户、博文标题、博文内容、点赞数、回复数、游览量。
    发布新博客
    对博客进行分类
    对博客打标签
    修改博客
    删除博客
    根据标题,分类,标签查询博客

管理博客分类 :分类ID、分类名称、分类别名、分类描述、父分类ID
    新增一个分类
    修改一个分类
    删除一个分类
    根据分类名称查询分类
    
管理标签 :标签ID、标签名称、标签别名、标签描述、父标签ID
    新增一个标签
    修改一个标签
    删除一个标签
    根据名称查询标签
评论管理 :评论ID、评论日期、发表用户、评论文章ID、评论内容
	删除
	修改
	
登录管理: 登录时间,登录ip,登录用户名,
 	删除
 	查找:按时间或ip查找

```
